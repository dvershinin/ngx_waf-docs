# 概述

因为模块的配置比较复杂，为了降低使用难度，我们在这里列出了一些常见用法。

  • 针对路径或文件限流
  • 开启验证码
  • 在被攻击时降低带宽占用
  • 抵御分布式 CC 攻击（分布式 HTTP 洪水）
  • 开启 ModSecurity
